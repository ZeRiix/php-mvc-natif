###### PMN (php mvc natif) est un framework mvc php compatibke avec une bdd MySQL

## Installation

installation avec docker

```bash
docker-compose up -d
```

## Config file

```bash
./docker-compose.yml
./docker/
./.env
./config/config.php
```
